Bind input to variable.
Simple binding
<input class="form-control" name="title" [(ngModel)]="simpleNgModelValue" required />

Value is dynamic: {{ simpleNgModelValue }}

Reference to input through ngModel
Model reference and properties.
<input class="form-control" name="title" ngModel required #InputReference="ngModel"//>

{{ InputReference.status }} - InputReference.status
{{ InputReference.touched }} - InputReference.touched

                        <form #myForm="ngForm" novalidate (ngSubmit)="onSubmit(myForm)">
                            <div>
                                <input
                                    name="name"
                                    #name="ngModel"
                                    ngModel
                                    required/>
                            </div>
                            <div>
                                <input
                                    type="password"
                                    name="password"
                                    #password="ngModel"
                                    ngModel
                                    required
                                    minlength="3">
                            </div>
                            <input type="submit" value="Submit!" [disabled]="myForm.invalid">
                        </form>
